# TripFlow360 - Enterprise Travel & Expense Data Platform

## Project Overview

TripFlow360 is an Enterprise Data Engineering platform designed to centralize travel and expense data from multiple business systems into a unified analytical platform.

The platform enables organizations to monitor travel requests, approvals, bookings, expenses, reimbursements, and vendor performance through automated ETL pipelines and centralized reporting.

---

## Business Problem

The organization stores travel-related information across multiple systems.

- HRMS
- Travel Portal
- Flight Booking System
- Hotel Booking System
- Cab Booking System
- Expense Management System
- Finance ERP

Since the data is scattered across different applications, management cannot obtain a unified view of travel spending and employee travel behavior.

---

## Objective

Build a centralized Enterprise Data Platform capable of

- Collecting data from multiple systems
- Cleaning and validating data
- Loading data into Azure Data Lake
- Transforming data using Azure Databricks
- Loading analytical data into Azure SQL Database
- Supporting dashboards and reporting

---

## Expected Outcome

A scalable and production-style Data Engineering platform capable of supporting business analytics and reporting.
